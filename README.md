### Core Gameplay
- **Customizable Lizard Character**: Choose colors, patterns, and accessories
- **Progressive Levels**: Increasing difficulty with new challenges
- **Leaf Collection**: Collect various types of leaves with different point values
- **Boss Battles**: Face challenging boss leaves with multiple health points
- **Power-ups**: Speed boosts, point multipliers, and magnetic collection
- **Achievement System**: Unlock achievements and track your progress

### Gallery & Progress
- **Collection Gallery**: View your collected leaves, power-ups, and achievements
- **Progress Tracking**: Monitor your collection progress (1000 leaves = 100% completion)

## üöÄ Quick Start

### Playing the Game
1. **Download** the `lizard_game.html` file
2. **Open** it in any modern web browser (Chrome, Firefox, Safari, Edge)
3. **Enjoy** playing immediately - no installation needed!

### For Developers
```bash
# Clone the repository
git clone https://github.com/stevebuscemi1/lizard_game.git
cd lizard-game

# Open the game in your browser
# Simply open lizard_game.html or use a local server
python -m http.server 8000  # Python 3
# python -m SimpleHTTPServer 8000  # Python 2
# Then visit http://localhost:8000
```

## üéØ How to Play

### Controls
- **Desktop**: Use mouse/touchpad to move the lizard
- **Mobile**: Touch and drag to control movement
- **Menu Navigation**: Click/tap buttons to navigate menus

### Objective
1. **Customize** your lizard with colors, patterns, and accessories
2. **Collect** all leaves in each level to progress
3. **Avoid** obstacles like rocks
4. **Use** power-ups strategically for higher scores
5. **Defeat** boss leaves by tapping them multiple times
6. **Complete** levels and unlock achievements

### Tips
- Build combo multipliers by collecting leaves quickly
- Use power-ups strategically for maximum points
- Explore different customization options
- Check the gallery to track your collection progress
